# 八字排盘

一个本地网页程序，支持输入出生日期、时间、时区后生成八字四柱，并查看大运、流年、流月、常见神煞和案例记录。

## 功能

- 四柱排盘：年柱、月柱、日柱、时柱。
- 日期输入：支持公历出生时间，也支持农历年月日、闰月和时分输入。
- 节气边界：按立春定年、按节气定月。
- 细节展示：藏干、十神、纳音、五行分布、月令提示。
- 大运参考：按传统规则估算顺逆和起运时间，并在原盘旁选择大运。
- 流运排盘：在原盘旁横向选择大运、流年，并通过下拉选择流月。
- 案例记录：本地保存当前排盘输入、四柱摘要和备注，可一键载入或删除。
- AI 分析：通过本地服务端转发到 OpenRouter，结合当前所选大运、流年、流月生成中文分析。
- 神煞展示：天乙、太极、文昌、禄神、羊刃、桃花、驿马、华盖、将星、红鸾、天喜、国印、金舆、空亡等常用神煞。
- 数据导出：可复制完整排盘 JSON。

## 运行

```bash
node server.mjs
```

打开：

```text
http://localhost:5173
```

AI 分析需要在 `.env` 中配置：

```text
OPENROUTER_API_KEY=你的 OpenRouter API Key
OPENROUTER_MODEL=openai/gpt-oss-120b:free
```

## 排盘规则

- 年柱和月柱按节气划分。
- 日柱按公历日期的儒略日干支循环计算。
- 默认启用 23 点子时换日，可在界面中关闭。
- 流月以目标年份立春起排十二个节气月，到下一年小寒止。
- 神煞采用常用速查规则：天乙等按日干/年干取，桃花驿马华盖将星按年支与日支三合局取，红鸾天喜按年支取，空亡按日柱旬空取。
- 不处理真太阳时校正，不同命理流派可能存在细节差异。

本工具仅供传统文化学习与排盘参考，不应作为医疗、法律、投资或重大人生决策依据。
